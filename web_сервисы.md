# Web-сервисы

Web-сервис --- это программный компонент, обеспечивающий взаимодействие
между приложениями через сеть.

## Основные особенности

-   Обмен данными по стандартным протоколам (HTTP, HTTPS)

-   Независимость от языка программирования

-   Использование форматов данных: JSON, XML

## Типы Web-сервисов

-   SOAP

-   REST

## Назначение

-   Интеграция различных систем

-   Обмен данными между приложениями

-   Создание распределённых систем

## Примеры использования

-   Онлайн-платежи

-   Авторизация через социальные сети

-   Получение данных погоды, курсов валют
