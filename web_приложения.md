# Веб-приложения

Web-приложения --- приложения, доступные пользователю через веб-браузер.

## Характеристики

-   Работают по клиент-серверной архитектуре

-   Не требуют установки на компьютер пользователя

-   Обновляются централизованно

## Преимущества

-   Кроссплатформенность

-   Доступность из любой точки мира

-   Масштабируемость

## Технологии

-   HTML, CSS, JavaScript

-   PHP, Python, Java

-   Framework'и: React, Angular, Django, ASP.NET
