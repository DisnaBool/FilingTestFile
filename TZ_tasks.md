# ТЗ к задачам

## Задача 1: Программа «Конвертер часов в минуты»

**Назначение:** Преобразование времени из часов в минуты.\
**Входные данные:** Целое число H (количество часов).\
**Выходные данные:** Целое число M (количество минут).\
**Формула:** M = H \* 60.\
**Пример:** H = 2 → M = 120.

------------------------------------------------------------------------

## Задача 2: Программа «Сравнение двух чисел»

**Назначение:** Определение наибольшего из двух различных чисел.\
**Входные данные:** Два числа A и B (A ≠ B).\
**Выходные данные:** Сообщение вида «Число \[X\] больше».

**Логика:** 1. Если A \> B, то X = A. 2. Иначе X = B.

**Пример:** A = 5, B = 8 → Сообщение: «Число B больше».

------------------------------------------------------------------------

## Задача 3: Программа «Допуск к экзамену»

**Назначение:** Проверка условия допуска студента к экзамену на основе
его текущей оценки.\
**Входные данные:** Целое число G (оценка за семестр).\
**Выходные данные:** Строка «Допущен» или «Не допущен».

**Логика:** 1. Если G \>= 3, статус --- «Допущен». 2. Иначе статус ---
«Не допущен».

**Пример:** G = 4 → «Допущен».

------------------------------------------------------------------------

## Задача 4: Программа «Расчет стоимости со скидкой»

**Назначение:** Вычисление итоговой стоимости покупки с применением
скидки при превышении лимита.\
**Входные данные:** Число P (цена за единицу), целое число K
(количество).\
**Выходные данные:** Число S (итоговая сумма к оплате).

**Правила:** 1. Рассчитать общую сумму: Total = P \* K. 2. Если Total \>
1000, применить скидку 10%: S = Total \* 0.9. 3. Иначе: S = Total.

**Примеры:** - P = 200, K = 6 (Total = 1200) → S = 1080. - P = 100, K =
5 (Total = 500) → S = 500.

------------------------------------------------------------------------

## Задача 5: Программа «Решение квадратного уравнения»

**Назначение:** Решение уравнения вида ax² + bx + c = 0 с полным
анализом всех возможных случаев.\
**Входные данные:** Три вещественных числа: a, b, c.\
**Выходные данные:** Сообщение с результатом вычислений.

**Логика работы:** 1. Если a = 0, вывести: «Уравнение не является
квадратным». Конец. 2. Вычислить дискриминант: D = b² - 4ac. 3. Анализ
корней: - Если D \< 0: «Корней нет». - Если D = 0: x = -b / (2a), «Один
корень: x». - Если D \> 0: x1 = (-b + √D) / (2a),\
x2 = (-b - √D) / (2a),\
«Два корня: x1, x2».

**Примеры:** - 1, -4, 4 → «Один корень: 2» - 1, 1, 1 → «Корней нет» - 0,
2, 5 → «Уравнение не является квадратным»
