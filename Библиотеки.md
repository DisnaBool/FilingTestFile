# Библиотеки

Библиотека --- это набор готовых функций, классов или модулей,
предназначенных для повторного использования в программах.

## Назначение библиотек

-   Ускорение разработки

-   Повышение качества и надёжности кода

-   Повторное использование решений

## Виды библиотек

-   Стандартные (встроенные в язык)

-   Сторонние (подключаемые отдельно)

-   Статические и динамические

## Примеры

-   Standard Library (C++)

-   .NET Framework

-   NumPy (Python)

-   jQuery (JavaScript)
