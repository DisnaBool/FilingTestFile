# Память в C++: стек и куча

В C++ есть два основных типа памяти: стек и куча.

## Стек (stack)

Стек --- это память, которая управляется автоматически.

Особенности: - быстро работает - хранит локальные переменные - очищается
автоматически после выхода из функции - ограниченный размер

Пример:

``` cpp
void func() {
    int x = 10; // в стеке
}
```

## Куча (heap)

Куча --- это динамическая память.

Особенности: - выделяется вручную (new) - освобождается вручную
(delete) - больше по размеру - медленнее стека

Пример:

``` cpp
int* p = new int(5); // выделение
delete p; // освобождение
```

## Разница

Стек: - автоматическое управление - быстрее - маленький размер

Куча: - ручное управление - медленнее - большой размер
