# Указатели в C++

Указатель --- это переменная, которая хранит адрес другой переменной.

## Объявление указателя

``` cpp
int a = 10;
int* p = &a;
```

p хранит адрес a.

## Разыменование

``` cpp
cout << *p;
```

-   --- получить значение по адресу.

## Основные операции

& --- взять адрес\
\* --- получить значение

## Пример

``` cpp
int a = 5;
int* p = &a;

*p = 10; // изменили значение a
```

## nullptr

Указатель может никуда не указывать:

``` cpp
int* p = nullptr;
```

## Важно

-   не использовать неинициализированные указатели
-   всегда освобождать память (если new)
