# Консольные приложения

**Консольные приложения** — программы, работающие в текстовом режиме через командную строку.

---

## Характеристики

- взаимодействие с пользователем через ввод и вывод текста
- отсутствие графического интерфейса
- простота разработки и отладки
- низкие требования к ресурсам

---

## Преимущества

- быстрая разработка
- удобны для обучения основам программирования
- часто используются для служебных утилит и скриптов

---

## Примеры языков и технологий

- C, C++
- Python
- Java
- C#
